# 📊 Zerodha Stock Trading Database System (SQL Project)

## 🏢 Project Overview

This project is a SQL-based database design inspired by Zerodha, India’s leading stock trading platform. It simulates real-world trading operations such as user management, order execution, portfolio tracking, and financial transactions.

## 🎯 Objective

To design a scalable and efficient relational database system that handles:

* User accounts
* Stock trading
* Orders & trades
* Portfolio management
* Financial transactions

## 🚀 Key Features

* Discount brokerage model simulation
* Real-time trading structure (orders & trades)
* Portfolio tracking system
* Transaction management (deposit/withdrawal)
* Data integrity using primary & foreign keys

## 🧩 Database Schema

The system includes the following entities:

* Users
* Accounts
* Stocks
* Orders
* Trades
* Portfolio
* Transactions

## 🔗 Relationships

* One user → multiple accounts
* One account → multiple orders
* One order → multiple trades
* One user → multiple portfolio records
